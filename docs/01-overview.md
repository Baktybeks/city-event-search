# Обзор проекта

## Назначение

**City Event Search** — веб-приложение для поиска и управления городскими событиями: концерты, выставки, фестивали, мастер-классы, спорт, конференции и др.

Реализовано:

- Просмотр каталога событий и поиск по категориям, дате, месту, тексту
- Регистрация и авторизация пользователей
- Роли: **гость**, **пользователь**, **организатор**, **администратор**
- Избранное и регистрация на события
- Панель организатора: создание/редактирование событий, аналитика
- Админ-панель: пользователи, модерация событий, настройки

## Стек технологий

| Категория | Технологии |
|-----------|------------|
| **Frontend** | Next.js 15, React 19, TypeScript, Tailwind CSS 4 |
| **Backend / BaaS** | Appwrite (Auth, Database, Storage) |
| **Состояние** | Zustand (авторизация), TanStack React Query (данные) |
| **UI** | Headless UI, Lucide React, React Toastify, Chart.js, react-datepicker |

## Требования

- **Node.js** 20 или выше
- **Аккаунт Appwrite** — [cloud.appwrite.io](https://cloud.appwrite.io) или свой сервер

## Основные возможности

- Публичный каталог событий с фильтрами и пагинацией
- Детальная страница события с организатором, регистрацией, избранным
- Личный кабинет: профиль, избранное, «мои события»
- Организатор: создание/редактирование событий, черновики, публикация, аналитика
- Админ: активация пользователей, управление событиями (публикация/отмена), аналитика
